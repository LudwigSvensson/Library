
# LibraryAngular
Detta projekt är en Angular-applikation som fungerar som ett användargränssnitt (UI) för att kommunicera med ett C# API. Det använder sig av HttpClientModule för att hämta och skicka data till API
, samt Syncfusion-komponenter för att visa data i ett visuellt tilltalande format.

## Funktioner
Datahämtning och -skickning: Använder Angulars HttpClientModule för att kommunicera med ett externt API.
Dashboard: Visar data hämtad från API
i form av tabeller och grafer med Syncfusion-komponenter.
Routing: Stöder navigering mellan olika vyer i applikationen med hjälp av Angulars routing.

## Teknologier
Angular: Ett frontend-ramverk för att bygga dynamiska webbapplikationer.
Syncfusion: Ett bibliotek med UI-komponenter för att visualisera data.
C# API: Backend som fungerar som datakälla för Angular-applikationen.
HttpClientModule: Angular-modul som används för att göra HTTP-anrop till API
